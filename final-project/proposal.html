Overview

Purpose

The app is provides weather information for different locations inputed by users. It provides a 7 day weather forecast.

Audience

The audience includes outdoor enthusiasts, hikers, travellers and farmers that would like to know the weather conditions to enable them plan their activities.

Major functions

  • Provides weather condition of the user's location on the home page
  • Provides a seven day forecast of the locations inputed by the user
  • Provides weather alerts provided by meteorological agencies.
